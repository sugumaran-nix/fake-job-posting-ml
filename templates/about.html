Fake Job Posting Prediction

An automated three-layer fraud detection system combining Natural Language Processing, Machine Learning classification, and an automated URL fraud detection engine. The system analyses job description text and independently evaluates company website domains using 10 rule-based checks — covering domain entropy, SSL validity, HTTPS usage, suspicious TLDs, free hosting platforms, typosquatting, subdomain depth, scam keywords, digit density, and domain length anomalies.

URL Fraud Detection Engine

1. HTTPS Check SSL/TLS encryption validation
2. IP Detection Raw IP address instead of domain
3. Free Hosting 26 known free hosting platforms
4. TLD Risk Score 20 suspicious + 10 trusted TLDs
5. URL Entropy Shannon entropy for randomness detection
6. Keyword Score 22 scam URL keywords matched
7. Subdomain Depth Excessive nesting detection
8. Typosquatting 13 brand domains checked
9. Digit Density Auto-registered domain pattern
10. Length Anomaly Too short / too long domain name
